laboratorio-de-midias

Sofware desenvolvido com o intuito de obtenção de nota parcial da disciplina de Linguagem de Programação para a Web (LpWeb) do Ceulp/Ulbra. Este, por sua vez, é responsável por capturar áudio, vídeo e fotografar utilizando a câmera do seu pc, para tanto, foi feito uso de bibliotecas do HTML5.
